# Proffy
> Proffy mobile | Next Level Week 2 - Rocketseat.


## Installation

```bash
# Clone this repository
$ git clone https://github.com/hugomatheus/nlw2-mobile-proffy.git

# Go into the repository
$ cd nlw2-mobile-proffy

# Install dependencies
$ yarn install

# Start expo
$ yarn start
```

## Technologies

* Expo
* React Native
* Axios
